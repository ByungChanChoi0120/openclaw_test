# README_WORKSPACE.md

이 워크스페이스에서 자주 보는 파일을 빠르게 찾기 위한 인덱스.

## 핵심 파일
- `AGENTS.md` : 운영 원칙/세션 행동 규칙
- `SOUL.md` : 말투/정체성
- `USER.md` : 사용자 선호/배경
- `MEMORY.md` : 장기 기억(큐레이션)
- `WORKFLOW_AUTO.md` : 위임 기준, task 작성 규칙, 품질 게이트

## 운영 로그
- `memory/YYYY-MM-DD.md` : 일일 로그(결정/실수/교훈)

## 템플릿
- `templates/spawn-task.md` : sessions_spawn 작업 지시 템플릿

## 추천 운영 루틴 (짧게)
1. 작업 시작 전: 목표/완료기준 1~2줄 확정
2. 위임 전: `WORKFLOW_AUTO.md` 기준으로 위임 가능 여부 판단
3. 작업 후: `memory/YYYY-MM-DD.md`에 변경 이유 한 줄 기록
4. 큰 변경 후: git 커밋으로 스냅샷 남기기
